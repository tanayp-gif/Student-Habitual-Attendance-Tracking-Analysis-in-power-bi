# Student-Habitual-Attendance-Tracking-Analysis-in-power-bi
The project titled “Student Habitual Attendance Tracking Analysis in Power BI” focuses on analyzing student attendance patterns to identify habitual absenteeism and improve academic performance. The dataset includes student attendance records over a period of time, capturing details such as student ID, class, date, and attendance status. Using Microsoft Power BI, the data is cleaned, transformed, and visualized through interactive dashboards and reports.

Various visualizations such as bar charts, line graphs, and pie charts are used to track attendance trends, monthly performance, and individual student behavior. Key performance indicators (KPIs) like attendance percentage, absentee rate, and consistency scores are calculated to identify students with irregular attendance patterns. Filters and slicers allow users to analyze data based on class, time period, or specific students.

The analysis helps in early identification of habitual absentees, enabling timely intervention by educators. In conclusion, this project demonstrates how data analytics and visualization tools like Power BI can support decision-making in educational institutions and improve student engagement and performance.
